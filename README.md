# Edugami - Plataforma

## Guía de variables de entorno

> **Nota**: Ver [.env.example](.env.example) para mejor entendimiento.

```
NEXT_PUBLIC_API_URL = "url de la API"
NEXT_PUBLIC_AR_URL = "url de la página de AR"
```

## Pasos para desplegar proyecto de next js

Primero compilamos el proyecto con el comando `npm run build` para luego ejecutar el servidor con `npm start`.
